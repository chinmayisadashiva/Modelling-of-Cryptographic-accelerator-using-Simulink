# Modelling-of-Cryptographic-accelerator-using-Simulink

Designed and modeled an Elliptic Curve Cryptography (ECC)-based cryptographic accelerator for secure key generation, encryption, and decryption. The project focuses on hardware-oriented cryptographic design, secure communication, and performance analysis for embedded systems.

## Overview

This project focuses on the design and modeling of a Cryptographic Accelerator based on Elliptic Curve Cryptography (ECC). The objective is to demonstrate secure key generation, encryption, and decryption while exploring hardware-oriented cryptographic architecture for embedded systems. The project emphasizes secure communication, efficient cryptographic processing, and the application of ECC for resource-constrained devices.

## Features

- Secure key generation using Elliptic Curve Cryptography (ECC)
- ECC-based encryption and decryption process
- Hardware-oriented cryptographic architecture
- Secure communication workflow
- Performance analysis of cryptographic operations
- Modular design suitable for embedded system applications

## Technologies Used

- Simulink
- Verilog HDL
- LTSpice
- Elliptic Curve Cryptography (ECC)
- Digital Design

## Methodology

The project follows the below workflow:
1. Key Generation
2. Plaintext Input
3. ECC Encryption
4. Ciphertext Generation
5. ECC Decryption
6. Plaintext Recovery
7. Performance Evaluation

## Results

The project demonstrates:
- Design and modeling of an ECC-based cryptographic accelerator.
- Secure key generation, encryption, and decryption workflow.
- Hardware-oriented cryptographic architecture for embedded systems.
- Performance evaluation of cryptographic operations.
- Efficient and secure communication using Elliptic Curve Cryptography.
